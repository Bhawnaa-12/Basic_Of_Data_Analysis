# Data Analysis with NumPy, Pandas & Matplotlib 📊

## Overview

This Python script demonstrates the fundamentals of data analysis and visualization using three powerful libraries: NumPy, Pandas, and Matplotlib. It covers numerical operations, data handling, and graphical representation of data.

## Features

* Perform numerical computations using NumPy
* Create and manipulate datasets using Pandas
* Visualize data using different types of charts with Matplotlib
* Demonstrates real-world data analysis workflow

* Python 3.x

## How It Works

### 1. NumPy (Numerical Operations)

* Creates arrays
* Performs operations like:

  * Mean
  * Sum
  * Maximum and Minimum
* Reshapes arrays into 2D format

### 2. Pandas (Data Handling)

* Creates a DataFrame with student data
* Displays:

  * Head (first few rows)
  * Info (data types and structure)
  * Statistical summary
* Filters data (e.g., students with marks > 85)
* Adds a new column (`Grade`)

### 3. Matplotlib (Visualization)

Generates multiple plots:

* 📈 Line Chart (Marks vs Students)
* 📊 Bar Chart (Marks comparison)
* 🔵 Scatter Plot (Age vs Marks)
* 📉 Histogram (Marks distribution)
